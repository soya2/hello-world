Java的一些前置工作


Friday, July 26th 2019, 3:14 pm

必备下载

首先要先下载JDK，它是Java开发的软件工具包，是必要的文件，请先下载这个：
Java SE Downloads: (https://www.oracle.com/technetwork/java/javase/downloads/index.html)
点击网页中央下方有”Java Platform (JDK) 12”描述的的方块图形，然后在下方的这个区域点击涂黄的部分，然后下载
*”Windows 158.63 MB jdk-12.0.2_windows-x64_bin.exe”*或者
*”Windows 179.57 MB jdk-12.0.2_windows-x64_bin.zip”*中的其中一个,随后安装即可。

环境变量配置

按操作依次打开窗口；
控制面板中的系统，然后点击左侧涂色的高级系统设置

随后在弹出的”高级”选项卡中的最下方点击涂色的环境变量

然后打开的窗口分为用户变量和系统变量上下两个部分，首先对系统变量部分进行操作；首先查看变量中是否有名为*”JAVA_HOME”*的变量，如果没有就点击新建按钮，若有请点击编辑按钮。变量名填写为*”JAVA_HOME”*，变量值填写为你安装JDK*文件夹*的位置，点击确定。

之后对用户变量进行操作，选择Path变量后点击编辑，新建两个变量值%JAVA_HOME%\bin和%JAVA_HOME%\jre\bin，点击确定。

变量设置完成后，启动CMD直接输入java -version或者java*均有反应，而不是“‘java’ 不是内部或外部命令，也不是可运行的程序”*即可。
