新标日本语初级上

发表于 2020-06-26 更新于 2020-07-07

初级上

第一课

  • 李さん　は　中(ちゅう)国(ごく)人(じん)　です。
    小李 是 中国人
  • 森さん　は　学(がく)生(せい)　でわ（じや）　ありません。
    森先生 不是 学生
  • -森さん　は　学生　ですか。
    森先生 是 学生 吗

-はい　そうです。
-はい　学生です。
是，是学生
-いいぇ　ちがいます。
-いいぇ　学生では　ありません。
不，不是

  • 李さん　は　JC企(き)画(かく)　の　社(しゃ)員(いん)　です
    小李 是 JC策划公司 的 职员
  • 人称
    第一人称わたし
    第二人称あなた
    第三人称なの　人(ひと)
    称呼他人加さん
  • 日语中，在上下文能明白说的是谁时，可不加第一二人称，不加人称时显得礼貌
  • 自我介绍结束后可说请多关照

はじめまして、　李です。　どうぞ　よろしく　お願(ねが)いします。
初次见面，我姓李。请多关照。
こちらこそ、　どうぞ　よろしく　お願いします。
我才是要请你多关照。

  • 会(かい)社(しゃ)員(いん)　和　社員
    会社員 表示一种职业，意思是公司职员
    社員 仅表示属于某一个公司的人

第二课

  • これ　は　本(ほん)　です。

这是 书（距离说话人较近/较近）
それ　は　鞄(かばん)　です。
那是 包（距离听话人较近/较远）
あれ　は　テレビ　です。
那是 电视机（据两者都较远/更远）

  • それ　は　何(なん)　ですか。
    那是 什么
    あの　人は　だれ（どなた(礼貌)）　ですか。
    那个 人是谁
  • その　自(じ)転(てん)車(しゃ)は　森さんのです。
    那辆 自行车 是森先生的
  • 森さんの　かばんは　どれ　ですか。
    森先生的包是哪个
    小野さんの　机(つくえ)は　どの机　ですか。
    小野女士的桌子是哪一张
    三个以上事物的时候用どれ/どの，单独使用时用どれ，修饰名词时用どの
  • 出于对长辈、上司等尊敬的对象，将 あの　人 中的人替换为　方(かた)
    何(なん)歳(さい)ですか　-おいくつですか(礼貌) 两种询问年龄的方法

第三课

  • ここ　は　デパート(Department store)　です

这里是百货商店
そこ　は　図(と)書(しょ)館(かん)　です
那里是图书管
あそこ　は　入(い)り口(くち)　です
那儿是入口

  • 小野さん　は　事(じ)務(む)所(しょ)　です
    小野女士在事务所，注意这里不能译为小野是事务所，です比汉语的是含义更加广泛
  • あなたの　かばんは　どこですか
    你的包在哪儿？
    上述提问位置的词语的礼貌用法分别为こちら/そちら/あちら/どちら,在问国家时礼貌用法为お国(くに)は
  • 張さん　も　中国人です
    小张也是中国人
  • 今(きょ)日(う)は　水(すい)曜(よう)日(び)ですか，木(もく)曜日ですか
    今天是周一还是周四？
  • これは　いくらですか
    这个多少钱
    あれは　(いくらですか)可省略
    那个多少钱？一般在两个问句后面相同时，可省略后面部分，只说あれは即可，但须升调

第四课

  • 部(へ)屋(や)　に　机　が　あります
    房间里有张桌子
    公(こう)園(えん)　に　猫(ねこ)　が　います
    公园里有一只猫
    あります常指无意志的事物，而います常指有意志的事物，如人，动物等
  • 本　は　ここ　に　あります
    书在这里
    いす　は　部屋　に　あります
    椅子在房间里
    いぬ(犬) は　どこ　に　いますか(与前面一样加 か 为疑问句)
    狗在哪里？
  • 時(と)計(けい) と　眼(め)鏡(がね)
    钟表和眼镜
  • 売(ぱい)店(てん)は　駅(えき)の　外(そと)　に　あります
    小卖部在车站的外面

上(うえ)　下(した)　前(まえ)　後(うし)ろ　隣(となり)　中(なか)　外(そと)
具体位置时用名词 + の + 位置，且の不能省略

  • 教(きょう)室(しつ)に　だれも　いません
    教室里谁也没有
    冷(れい)蔵(ぞう)庫(こ)に　何(なに)も　ありません
    冰箱里什么也没有
  • 问及亲属时礼貌用法为ご家(か)族(ぞく)、ご兄(きょう)弟(だい)、ご両(りょう)親(しん)
    虽写作兄弟，但兄弟、兄妹、姐弟、姐妹间都称 きょうだい
    离开家一个人生活称作　一(ひ)人(とり)暮(ぐ)らし

第六课

  • 吉田さんは　中国　へ　行(い)きます
    吉田先生去中国，へ是表移动目的助词
    森さんは　日本　へ　帰(かえ)ります
    森先生回日本
    李さんは　どこ　へ　行きましたか
    小李去哪里了
  • 李さんは　先(せん)月(げつ)　北京　から　来(き)ました
    小李上个月从北京过来
  • 小野さんは　友(とも)達(だち)　と　帰りました
    小野女士和朋友一块儿回去了
  • 上海まで　飛(ひ)行(こう)機(き)　で　行きます
    到上海坐飞机去，用交通工具时用助词で
    李さんは　歩(ある)いて　アパート(Apartment)　へ　帰りました
    小李步行回到公寓，不用交通工具时用歩いて
  • 森さんは　東京　から　広島　まで　新(しん)幹(かん)線(せん)　行きます
    森先生从东京乘坐新干线到广岛去
  • わたしの　部屋には　電話が　ありません
    我的房间里没有电话
    韓国へは　行きました。　中国へは　行きませんでした
    我去了韩国，没有去中国

に/で/へ/から/まで/と + は
は既可以单独使用，也可以和上述助词构成一种复合形式

  • あの　人は　誰ですか。　-たしか　李さんの　会社の　人です
    那个人是谁？ -我记得好像是小李他们公司的人
  • うち　家()
    一般指建筑物/居住地时用 家； 着重指家人的时候用 うち
  • 今日は　まっすぐ　帰ります
    まっすぐ表示笔直的意思，与回家搭配时可表示哪也不去直接回家
  • 星期/日期表 P80

第七课

  • 李さんは　毎日　コーヒー(Coffee)　を　飲(の)みます
    小李每天喝咖啡
    李さんは　毎(まい)朝(あさ)　何　を　食(た)べますか
    小李每天早餐吃什么
    当动词有个指向的对象(宾语)时，对象后要加 を
  • 李さんは　図書館　で　勉(べん)強(きょう)します
    小李在图书馆学习
    今日　どこ　で　新聞　を　読(よ)みましたか
    今天，你在哪里看报纸了
    与第四课中的 に 功能类似，に 是指事务在某处，而 で 表示发生的动作在某处
  • わたしは　毎朝　パン(Bread)　か　お粥　を　食べます
    我每天早上吃面包或者稀饭
    休みは　月曜日　か　火曜日です
    我周一或周二休息
    名词间加 か 相当于或者的意思
  • この　本 を　ください
    我买这本书/请给我这本书
  • 关于 何(なん/なに) 的读法
    在量词和 の 前读 なん，其余读 なに，で 前两种都可以

第八课

  • 李さんは　日本語　で　手紙を　書きます
    小李用日语写信，で还可表示其他手段和原料
    手紙を　速達　で　送りました
    顺序问题？用速递寄了信
  • わたしは　小野さん　に　お土産　を　あげます
    我送小野女士礼物，あげます表示送，に表示接收者，を为物品助词
    小野さんは　森さん　に　チョコレート(chocolate)　を　あげました
    小野女士给了森先生巧克力
    通常在物品以（第一人称->二->三）或（三->三）形式移动时使用
    母は　長島さん に　ワイン(wine) を　あげました
    母亲送葡萄酒给了长岛先生
  • わたしは　小野さん　に　辞書　を　もらいました
    我从小野女士那儿得到一本词典/小野女士给了我一本词典
    わたしは　長島さん　から　写真　を　まらいました
    我从长岛先生那儿得到了照片
    母は　小野さんに　ハンカチ(handkerchief)を　もらいました
    小野女士送给母亲手绢
    まらいます和あげます相反，表物品以（三->二->一）或者（三->三）形式移动时使用
    相当于接受，得到等意思，に表示赠送者(也可以将赠送者视为物品移动的起点，所以可用から)，を为物品助词
  • 李さんは　明日　長島さん　に　会います
    小李明天见长岛先生
    わたしは　駅で　森さん　に　会います
    我在车站遇见了森先生
    会います即见面，对象助词用に
  • さっき　たった　今
    三者都表示离现在很近的过去，后面的动词一定用过去式，さっき比其余两者稍远一些
    而前に相当于“以前”的意思
  • ファックス(fox) メール(Email)
