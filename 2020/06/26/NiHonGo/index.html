新标日本语初级上

发表于 2020-06-26 更新于 2020-08-29

初级上

第一课

  • 李さん　は　中(ちゅう)国(ごく)人(じん)　です。
    小李 是 中国人
  • 森さん　は　学(がく)生(せい)　でわ（じや）　ありません。
    森先生 不是 学生
  • -森さん　は　学生　ですか。
    森先生 是 学生 吗

-はい　そうです。
-はい　学生です。
是，是学生
-いいぇ　ちがいます。
-いいぇ　学生では　ありません。
不，不是

  • 李さん　は　JC企(き)画(かく)　の　社(しゃ)員(いん)　です
    小李 是 JC策划公司 的 职员
  • 人称
    第一人称わたし
    第二人称あなた
    第三人称なの　人(ひと)
    称呼他人加さん
  • 日语中，在上下文能明白说的是谁时，可不加第一二人称，不加人称时显得礼貌
  • 自我介绍结束后可说请多关照

はじめまして、　李です。　どうぞ　よろしく　お願(ねが)いします。
初次见面，我姓李。请多关照。
こちらこそ、　どうぞ　よろしく　お願いします。
我才是要请你多关照。

  • 会(かい)社(しゃ)員(いん)　和　社員
    会社員 表示一种职业，意思是公司职员
    社員 仅表示属于某一个公司的人

第二课

  • これ　は　本(ほん)　です。

这是 书（距离说话人较近/较近）
それ　は　鞄(かばん)　です。
那是 包（距离听话人较近/较远）
あれ　は　テレビ　です。
那是 电视机（据两者都较远/更远）

  • それ　は　何(なん)　ですか。
    那是 什么
    あの　人は　だれ（どなた(礼貌)）　ですか。
    那个 人是谁
  • その　自(じ)転(てん)車(しゃ)は　森さんのです。
    那辆 自行车 是森先生的
  • 森さんの　かばんは　どれ　ですか。
    森先生的包是哪个
    小野さんの　机(つくえ)は　どの机　ですか。
    小野女士的桌子是哪一张
    三个以上事物的时候用どれ/どの，单独使用时用どれ，修饰名词时用どの
  • 出于对长辈、上司等尊敬的对象，将 あの　人 中的人替换为　方(かた)
    何(なん)歳(さい)ですか　-おいくつですか(礼貌) 两种询问年龄的方法

第三课

  • ここ　は　デパート　です

这里是百货商店
そこ　は　図(と)書(しょ)館(かん)　です
那里是图书管
あそこ　は　入(い)り口(くち)　です
那儿是入口

  • 小野さん　は　事(じ)務(む)所(しょ)　です
    小野女士在事务所，注意这里不能译为小野是事务所，です比汉语的是含义更加广泛
  • あなたの　かばんは　どこですか
    你的包在哪儿？
    上述提问位置的词语的礼貌用法分别为こちら/そちら/あちら/どちら,在问国家时礼貌用法为お国(くに)は
  • 張さん　も　中国人です
    小张也是中国人
  • 今(きょ)日(う)は　水(すい)曜(よう)日(び)ですか，木(もく)曜日ですか
    今天是周一还是周四？
  • これは　いくらですか
    这个多少钱
    あれは　(いくらですか)可省略
    那个多少钱？一般在两个问句后面相同时，可省略后面部分，只说あれは即可，但须升调

第四课

  • 部(へ)屋(や)　に　机　が　あります
    房间里有张桌子
    公(こう)園(えん)　に　猫(ねこ)　が　います
    公园里有一只猫
    あります常指无意志的事物，而います常指有意志的事物，如人，动物等
  • 本　は　ここ　に　あります
    书在这里
    いす　は　部屋　に　あります
    椅子在房间里
    犬(いぬ) は　どこ　に　いますか(与前面一样加 か 为疑问句)
    狗在哪里？
  • 時(と)計(けい) と　眼(め)鏡(がね)
    钟表和眼镜
  • 売(ぱい)店(てん)は　駅(えき)の　外(そと)　に　あります
    小卖部在车站的外面

上(うえ)　下(した)　前(まえ)　後(うし)ろ　隣(となり)　中(なか)　外(そと)
具体位置时用名词 + の + 位置，且の不能省略

  • 教(きょう)室(しつ)に　だれも　いません
    教室里谁也没有
    冷(れい)蔵(ぞう)庫(こ)に　何(なに)も　ありません
    冰箱里什么也没有
  • 问及亲属时礼貌用法为ご家(か)族(ぞく)、ご兄(きょう)弟(だい)、ご両(りょう)親(しん)
    虽写作兄弟，但兄弟、兄妹、姐弟、姐妹间都称 きょうだい
    离开家一个人生活称作　一(ひ)人(とり)暮(ぐ)らし

第五课

  • 今(いま)　何(なん)時(じ)　ですか
    现在是几点
    双方都明确讲现在的时间时，今 可以省略，只用 何時
    8時30分(さんじゅっ　ぶん)です/今　午前　8時半(はん)です
    现在是上午8点半
    30分可用半来替代，还可以加上 午前 或 午後
  • 森さんは　毎(まい)日(にち)　働(はたら)き　ます
    森先生每天工作
    田中さんは　明日(あした)　休(やす)み　ません
    田中先生明天不休息
    森さんは　先(せん)週(しゅう)　休み　ました（か）
    森先生上周休息了（吗）
    わたしは　昨日(きのう)　働き　ませんでした
    我昨天没有上班
    叙述现在或未来的习惯性动作、状态时，肯定时用ます，否定时用ません
    叙述过去的动作时，肯定时用ました，否定时用ませんでした
  • 学(がっ)校(こう)は　8時半　に　始まります
    学校8点半开始上课
    わたしは　明日　休みます
    我明天休息
    表示动作发生的时间，要在时间后加に，而今日/明日/去年/来年等词后不能加に
  • わたしは　9時　から　5時　まで　働きます
    我从九点开始到5点工作
  • 試(し)験(けん)は　いつ　始まりますか
    考试什么时候开始
    仕(し)事(ごと)は　何時に　終わりますか
    工作什么时候结束
    时间的问法比较抽象时用いつ，比较具体时用何時，同样要加に
  • 小野さんは　今日は　休みます⚡
    小野女士今天休息
    森さんは　毎朝　何時に　起きますか。－いつもは　7時ごろです
    森先生，你每天早上什么时候起床。-我一般是7点左右起床
