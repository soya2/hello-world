SQL常用命令即库操作

在终端中输入mysql -u[用户名] -p后弹出输入密码行，输入后进入mysql
查看当前库使用show databases;，注意在mysql中所有语句以;结尾
进入库使用use [库名];
查看库中表使用show tables;，在库中查看其它库表使用show tables from [库名];
查看表结构使用desc [表名];

查询

基础查询

查询某张表的单个或多个字段使用select [字段名1],[字段名2] from [表名];
查询所有字段使用select * from [表名];
用select方法也可查询常量、表达式和函数，如：

可对查询结果起别名，使用select [字段名1] as [别名1],[字段名2] as [别名2] from [表名];
用空格替代as效果一样，不过有关键字冲突的风险，select [字段名1] [别名1],[字段名2] [别名2] from [表名];
对于一字段中重复的对象，可用distinct去重，select distinct [字段名] from [表名]；
在sql中‘+’符号只能进行数值运算，若对象不是数值则尝试转换后运算，转换失败则视为0
若需要两字符拼接，则使用concat函数，select concat('字符1','字符2','字符3') from [表名];
若查询须拼接的字段已为字符型，参数可直接使用字段名，select concat([字段名1],[字段名2],[字段名3]) from [表名];

需要注意的是在做拼接或运算时遇到值为null的参数，结果一定为null，在使用中根据需要可进行一些变化
使用ifnull()函数可以在遇到null值时输出其他参数，不为null时输出原值，ifnull([字段名],[输出值]);

条件查询

sql中除了字段，也可按条件查询

类似于编程语言中的条件表达式，sql中的条件和逻辑运算符为：

  • > 大于
  • < 小于
  • <> 或 != 不等于
  • <= 大于等于
  • >= 小于等于
  • and 与
  • or 或
  • not 非

实例SELECT last_name,department_id FROM employees WHERE salary>12000;
sql中支持模糊条件的搜索

  • like 可模糊查询指定字符，同时支持通配符%(表示0或任意多个字符)和_(表示任意单个字符)，在指定字符和通配符冲突时使用转义字符\，查询字段中第三位为a的对象：select [字段名] from [表名] where [字段名] like '__a%';
  • between and 描述字段是否置于指定上限和下限之间，等价于>= 和 <=，and左右的两个值包含临界值且顺序不能颠倒，select [字段名] from [表名] where [字段名] between [上限] and [下限];
  • in 描述字段中是否有指定字符，等价于=，in中的指定字符不支持通配符，select [字段名] from [表名] where [字段名] in('字符1','字符2');
  • is null 由于=和!=不能判断null值，故采用is null或is not null，select [字段名] from [表名] where [字段名] is null;

    sql中存在一个名为安全等于的符号<=>，它可以直接判断是否等于null或普通数值，但可读性较差
    select [字段名] from [表名] where [字段名] <=> [null或普通数值];

分组函数

将一组数据进行处理后输出一个值，类型严格，常见的函数有：

  • sum(字段)，求和
  • avg(字段)，求平均值
  • min(字段)，最低
  • max(字段)，最高
  • count(字段)，个数 #count()和count(1)都可以查行数，但1的效率比\较高
    可配合distinct做去重操作
